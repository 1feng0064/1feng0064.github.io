  2. Qus: Define version control and give one example.

    Ans: Version control is a system which can help people to track changes they make in the code over time. As people edit their code, they ask the version control system to take a snapshot of the files. The version control system saves records permanently therefore, people can recall the records later when they need them.
    Example : <tag> An example of version control is GitHub. </tag>
